# Traffic Light Control

### Introduction

You must provide the code for an application that simulates a set of traffic lights at an intersection.

Traffic lights are designated as (N,S) and (W,E) just like a compass




### How to use

#### Command line

```command
$ git clone https://github.com/greeneley/traffic-light.git
$ cd traffic-light
$ node traffic-light.js
```

How to install node ?

 https://nodejs.org/en/download/package-manager/

### Tech-stack

* HTML
* CSS
* Javascript
